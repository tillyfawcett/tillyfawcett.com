PROJECTS

SWIPE-CHEF

Think Tinder, but for indecisive foodies. Swipe-Chef streamlines the search for recipes, simply swipe or use your keyboard to move through recipe cards until you find the perfect match. Swipe-Chef was developed and designed using HTML, CSS and JavaScript by a team of Le Wagon graduates.


BARLY

Queueing sucks. Queueing for a drink on a Friday night after a long week sucks even more. Barly is a web application that allows users to order their drinks from the comfort of their table - avoiding the bustle of a queue. Barly is a group final project from Le Wagon's 9 week bootcamp, built using Ruby on Rails, HTML, CSS/SASS, JavaScript, jQuery, Bootstrap, SQL and MVC Architecture.


CAMEL SKI SCHOOL

Where it all began.. I redesigned a clean commercial website for Camel Ski School using SquareSpace, after finding the original WordPress site ugly, impractical and difficult to navigate for the user. This is what led me to pursue a career in coding.
